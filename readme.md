TFS
===

Description
-----------

This package is used to provide an abstraction to the normal profile that is loaded for your bash terminal. The purpose is to allow for a consolidated place to configure and extend the shell.   


### Install 
```
$ ./tsh install

```

#### Updates
~/.profile or ~/.bashrc
